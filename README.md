# 📚 Gestion de Bibliothèque

Application web développée par **Hadil Jobrani** avec Django.

## Description
Application de gestion de bibliothèque permettant d'ajouter, rechercher, modifier et supprimer des livres.

## Fonctionnalités
- ➕ Ajouter un livre (titre, auteur, prix, disponibilité)
- 📋 Afficher la liste des livres
- 🔍 Rechercher par titre ou auteur
- ✏️ Modifier un livre
- 🗑️ Supprimer un livre

## Technologies utilisées
- Python 3.14
- Django 4.2
- Bootstrap 5
- SQLite

## Installation
```bash
git clone https://github.com/hadiljobrani/gestion_bibliotheque.git
cd gestion_bibliotheque
python -m venv .venv (pour créer l'environement virtuelle )
ctr+shift+p (pour activer l'environement virtuelle)
pip install django==4.2
python manage.py migrate
python manage.py runserver
```
